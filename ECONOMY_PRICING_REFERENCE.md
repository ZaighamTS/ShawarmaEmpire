## 🔢 COST FORMULAS

### Upgrade Cost Formula

```
cost = (basePrice - prestigeReduction) × (level^multiplier) × (1 / (1 + level × 0.1))
```

**Components:**
- `basePrice`: Base cost (varies by type)
- `prestigeReduction`: `prestigeStars × 2.5`
- `level`: Current upgrade level (starts at 1)
- `multiplier`: Exponential multiplier (varies by type)
- `diminishingFactor`: `1 / (1 + level × 0.1)` - prevents astronomical costs

### Purchase Cost Formula (Core Buildings)

```
purchaseCost = (basePrice - prestigeReduction) × (3.5^existingCount)
```

**Components:**
- `basePrice`: Base cost (varies by type)
- `prestigeReduction`: `prestigeStars × 2.5`
- `existingCount`: Number of buildings already placed
- `scalingFactor`: 3.5x per additional purchase

### Purchase Cost Formula (Extra Buildings)

```
purchaseCost = (basePrice - prestigeReduction) × (3.5^existingCountOfType)
```

**Components:**
- `basePrice`: Base cost (varies by building type)
- `prestigeReduction`: `prestigeStars × 2.5`
- `existingCountOfType`: Number of buildings of same type already purchased
- `scalingFactor`: 3.5x per additional purchase of same type
